Getting Started: 42 Header

A “42 header” refers to a specific format and content that is required at the beginning of every source code file you write. It’s a standardised header that contains important information about the file and serves as a form of identification.

The 42 header typically includes the following elements:

  1. File Name: The name of the file, which usually corresponds to the function or purpose of the code.
  2. Author: Your username, along with your 42 email address, to identify who wrote the code.
  3. Creation Date: The date when the code was initially created.
  4. Last Updated: The date when the code was last modified.

Here is an example of 42 header:

VIM
/* ************************************************************************** */
/*                                                                            */
/*                                                        :::      ::::::::   */
/*   example.c                                          :+:      :+:    :+:   */
/*                                                    +:+ +:+         +:+     */
/*   By: nshahid <nshahid@student.42abudhabi.ae>    +#+  +:+       +#+        */
/*                                                +#+#+#+#+#+   +#+           */
/*   Created: 2023/05/27 12:07:37 by nshahid           #+#    #+#             */
/*   Updated: 2023/05/27 12:07:37 by nshahid          ###   ########.fr       */
/*                                                                            */
/* ************************************************************************** */
NSHAHID

At the 42 coding school, adhering to the specific format and content of the 42 header is typically mandatory as part of their coding standards and requirements.

This will act as a quick and easy guide on how to install the Standard 42 Header! Simply follow the steps:

  1. Step 1: Clone the Git Repository

Let’s kick things off by cloning the Git repository that houses the 42 Header. It’s as easy as typing this command into your terminal:

TERMINAL
git clone https://github.com/42Paris/42header.git 42header
NSHAHID

Feel free to replace “42header” with any name you fancy for the folder where you want to copy the repository.

  1. Step 2: Navigate to the Cloned Folder

Now that you’ve got the repository in your grasp, let’s venture into the world of the newly cloned folder. Just execute this command to change your directory:

TERMINAL
cd 42header
NSHAHID
  1. Step 3: Run the Shell Script

It’s time to bring out the big guns! Run the shell script that will install the plugin for the Standard 42 Header in Vim. Just type this simple command:

TERMINAL
./set_header.sh
NSHAHID
  1. Step 4: Check Your Username and Email

Before you dive into the coding adventure, let’s ensure your username and email address are perfectly set.

  • Open the “zshrc” file in your favorite text editor:
TERMINAL
vim ~/.zshrc
NSHAHID
  • Explore and find the following lines:
VIM
USER=username
export USER
NSHAHID

Make sure the “username” matches your login information on the system. This is crucial for every file you create during your 42 journey.

  • Oh, wait! We’re missing something important—the email address required for the Standard 42 Header. Let’s add the following lines:
VIM
MAIL=username@student.your42network
export MAIL
NSHAHID

Replace “username” with your login and “your42network” with the address of your school. For example, if you belong to 42 France, it would be “username@student.42.fr.” Not sure? Find your username and email on the 42 Intranet’s public profile.

  • Save and exit the editor!
  1. Step 5: Refresh and Get Set

One final touch, and you’re good to go! Refresh the “zshrc” file by using the source command in the terminal:

TERMINAL
source ~/.zshrc
NSHAHID

Now, brace yourself for the ultimate norminette experience!

  1. Step 6: Unleash the Power!

Open Vim and prepare to be amazed. Press the F1 key (or FN + F1, depending on your setup) and watch the Standard 42 Header appear like magic at the top of your code.

Congratulations! You’re now armed with the 42 Header. It’s time to embark on your coding odyssey.

Always keep in mind that the joyous norminette requires you to include the magnificent 42 Header at the beginning of every source file. Failure to do so might leave norminette feeling a bit grumpy. So, let’s spread the happiness by adding the 42 Header to every source file we create!
